# Mobile

## Instalando o expo

```
npm install expo
```

https://docs.expo.io/

## Criando o projeto

```
expo init mobile
```

## Iniciando a aplicação

```
npm start
```

Instale o aplicativo expo no celular e logo após leia o código QRCode que foi exibido na página http://localhost:19002/
